# Deployment Scripts

This directory contains deployment automation scripts.

## Available Scripts

- `deploy-l2.sh` - Deploy L2 rollup (if needed)
- `deploy-contracts.sh` - Deploy QRMS contracts to L2
- `verify-deployment.sh` - Verify deployed contracts

## Usage

```bash
# Deploy contracts
./scripts/deploy/deploy-contracts.sh

# Verify deployment
./scripts/deploy/verify-deployment.sh
```
