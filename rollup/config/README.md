# Rollup Configuration

This directory contains generated configuration files for the L2 rollup.

## Files

- `genesis.json` - L2 genesis block configuration
- `rollup.json` - Rollup configuration (chain ID, L1 contracts, etc.)
- `jwt.txt` - JWT secret for L2 engine authentication

## Generation

These files are automatically generated by `op-deployer` during the setup process.

## Location

The actual files are typically located in:
- `rollup/opstack/deployer/config/` (after deployment)
